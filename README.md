<h1> Paths Activity </h1>
Paths is a sliding tile game where the goal is to create closed paths. 64 tiles are to be arranged onto the 8Ã—8 grid, positioning the tiles so that paths are created. Paths activity can be integrated into classroom assignments. It helps develop logical thinking.
